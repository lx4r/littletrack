# LittleTrack

Minimal, offline-first, local time tracking PWA. The core value is simplicity, it's meant to be a quick fallback when primary time tracking tools aren't available.

## Tech Stack

Selection: PWA, TypeScript, Tailwind, pnpm, Biome (not ESLint/Prettier)

## Important commands

- `pnpm test:run` — Run all tests once
- `pnpm test:run MyFile` — Run specific test file
- `pnpm check` — Run linter and formatter checks
- `pnpm check:fix` — Fix linting and formatting issues
- `pnpm check-types` — Run TypeScript type checking
